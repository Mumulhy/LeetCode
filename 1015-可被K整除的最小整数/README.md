# 1015-可被K整除的最小整数

Author：_Mumu

创建日期：2025/11/25

通过日期：2025/11/25

*****

踩过的坑：

1. 增加1的数量，余数必循环，遇0则停止
2. 看题解才知道2和5的倍数必无解，非2和5的倍数必有解

已解决：502/4146

*****

给定正整数 k ，你需要找出可以被 k 整除的、仅包含数字 1 的最 小 正整数 n 的长度。

返回 n 的长度。如果不存在这样的 n ，就返回-1。

注意： n 可能不符合 64 位带符号整数。

 

示例 1：

输入：k = 1
输出：1
解释：最小的答案是 n = 1，其长度为 1。
示例 2：

输入：k = 2
输出：-1
解释：不存在可被 2 整除的正整数 n 。
示例 3：

输入：k = 3
输出：3
解释：最小的答案是 n = 111，其长度为 3。
 

提示：

1 <= k <= 105
